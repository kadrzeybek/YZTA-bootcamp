# Product Backlog

## Story 1: Ürün Bilgi Girişi Akışı (8 puan)
Satıcının ürün temel özelliklerini (materyal, tarz, renk, kullanım amacı) girebileceği basit bir giriş formu/akışı tasarlanır.

- [ ] Task: Girdi alanlarının belirlenmesi (materyal, tarz, renk, kategori vb.) — 2 puan
- [ ] Task: Form/giriş arayüzünün taslak (wireframe) çizimi — 3 puan
- [ ] Task: Girilen verinin JSON/veri modeli olarak tanımlanması — 3 puan

## Story 2: Tasarımcı Agent — Prompt Üretim Mantığı (13 puan)
Kullanıcı girdisini alıp Midjourney/DALL-E için görsel üretim promptu üreten agent mantığı kurulur.

- [ ] Task: Prompt şablonunun tasarlanması (ürün özelliği → prompt eşlemesi) — 5 puan
- [ ] Task: Örnek 5 ürün için prompt üretiminin test edilmesi — 5 puan
- [ ] Task: Üretilen promptların kalite kontrolü ve iyileştirilmesi — 3 puan

## Story 3: Tasarımcı Agent — Görsel Üretim Entegrasyonu (8 puan)
Üretilen promptun görsel üretim API'sine (Midjourney/DALL-E) gönderilip görselin alınması.

- [ ] Task: API/araç seçiminin yapılması (Midjourney vs DALL-E) — 2 puan
- [ ] Task: Basit bir API çağrısı ile örnek görsel üretiminin denenmesi — 5 puan
- [ ] Task: Üretilen görselin kaydedilmesi/gösterilmesi — 1 puan

## Story 4: Metin Yazarı Agent — Taslak Akış (8 puan)
Görsel ve ürün özelliklerini temel alarak SEO uyumlu ürün açıklaması üreten agent'ın ilk mantığının kurulması.

- [ ] Task: SEO uyumlu ürün açıklaması için prompt şablonunun yazılması — 5 puan
- [ ] Task: Örnek bir ürün için açıklama metni üretiminin test edilmesi — 3 puan

## Story 5: Takım & Süreç Kurulumu (5 puan)
- [ ] Task: GitHub reposunun kurulması ve README'nin oluşturulması — 1 puan
- [ ] Task: Backlog'un Miro/Trello'ya işlenmesi — 1 puan
- [ ] Task: Sprint board'un kurulması (To Do / In Progress / Done) — 1 puan
- [ ] Task: Daily Scrum kanalının/rutininin belirlenmesi — 1 puan
- [ ] Task: Roller ve sorumlulukların netleştirilmesi (PO, SM, Developer) — 1 puan

---
**Sprint 1 için hedeflenen toplam puan:** ~30-35 puan (Story 1, 2, 5 + Story 3'ün bir kısmı önerilir; Story 4 Sprint 2'ye kayabilir)
